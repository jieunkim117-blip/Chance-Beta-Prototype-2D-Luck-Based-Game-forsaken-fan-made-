# 🎮 Chance Beta Prototype

A 2D top-down browser-based game prototype featuring a high-risk, luck-based character system.

## 🧩 Overview
This project is a beta test focused on core gameplay mechanics such as movement, randomness, and ability stacking.

The player controls **Chance**, a character whose power depends entirely on luck.

## ⚡ Features
- WASD movement system
- Coin flip mechanic (risk vs reward)
- Charge & Weakness stack system
- Ability system:
  - Q: Coin Flip
  - E: One Shot
  - R: Reroll
  - T: Reset
- Visual feedback system (SUCCESS / FAIL / BOOM)
- Skill icons using custom images

## 🧪 Beta Scope
- No enemies
- Focus on core mechanics only
- Designed for testing and iteration

## 🛠️ Tech
- HTML
- CSS
- JavaScript (Vanilla)
